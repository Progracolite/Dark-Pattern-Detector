# Dark-Pattern-Detector
A front end only tool that can help identify manipulative text in websites. Built for our Webathon

**All processing is done in browser using Javascript.
**Patterns are scored to give a clear summary
**Visually pinpoints dark patterns and explains why it is manipulative

**HTML5: Structure
**CSS:Responsive designing using flexbox and css variables for theming.
**Javascript: We use an array of objects where each pattern is defined with its own regex,score and explanation.
