Dark Pattern Detector

How the app works

Paste text (webpage copy, cookie banners, or form text) into the input area, then click "Analyze Elements". The app scans the text for manipulative language patterns, highlights suspicious phrases, and produces a short summary and pattern list.

Reading the results

Summary shows the main findings, the highlighted box displays detected phrases inline, and the patterns list explains each manipulative element found with a brief description and a confidence indicator.
